Internship Project

Machine Learning Stock Price Prediction

This project combines financial data analysis, supervised machine learning, and an interactive Flask dashboard to forecast near-term closing prices from historical market data.

Project Overview

The application accepts a Yahoo Finance ticker symbol, downloads at least five years of daily OHLCV data, builds model-ready technical features, trains regression models, and visualizes historical performance alongside future price forecasts.

The dashboard includes company information, next-day prediction, 7-day and 30-day trend forecasts, moving averages, and recent historical price data.

Technologies Used

Python Flask HTML5 CSS3 Bootstrap 5 JavaScript Pandas NumPy Scikit-Learn yfinance Plotly

Machine Learning Approach

The model pipeline creates features such as previous close, moving averages, daily returns, short-term volatility, daily price range, and volume. The target variable is the next trading day's closing price.

The pipeline trains regression models (such as Linear Regression and Random Forest Regressor). Both models are evaluated under the hood on a time-ordered holdout set using R² Score, Mean Absolute Error, and Root Mean Squared Error, and the model with the stronger R² Score is automatically selected for forecasting.

Prediction Scope

The selected model predicts the next trading day directly, then generates 7-day and 30-day trends through iterative forecasting. Each future step refreshes derived features so the trend reflects the model's own projected path.

This application is built for academic demonstration and should not be used as financial advice.
